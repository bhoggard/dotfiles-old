# dotfiles-rcm

my dotfiles using Thoughbot's [rcm](https://github.com/thoughtbot/rcm)

## Install

Start by cloning this repo with

    git clone git@github.com:bhoggard/dotfiles.git ~/.dotfiles

then running

    env RCRC=$HOME/.dotfiles/rcm/rcrc rcup
